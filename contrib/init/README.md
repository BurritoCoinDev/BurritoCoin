Sample configuration files for:
```
SystemD: burritocoind.service
Upstart: burritocoind.conf
OpenRC:  burritocoind.openrc
         burritocoind.openrcconf
CentOS:  burritocoind.init
macOS:   org.burritocoin.burritocoind.plist
```
